Kebijakan Privasi

Catet Dulu · Terakhir diperbarui: 19 Juni 2026

Kebijakan privasi ini menjelaskan bagaimana aplikasi Catet Dulu menangani informasi pengguna. Dengan menggunakan aplikasi ini, Anda menyetujui kebijakan ini.

1. Data yang dikumpulkan

Aplikasi menyimpan data berikut di perangkat Anda:

  • Nama (opsional, untuk personalisasi tampilan)
  • Penghasilan bulanan dan persentase tabungan
  • Catatan pengeluaran (nominal, kategori, dompet, tanggal, catatan)

2. Cara penyimpanan data

Semua data disimpan secara lokal di perangkat Anda menggunakan penyimpanan internal aplikasi. Kami tidak mengoperasikan server untuk menyimpan data keuangan pribadi Anda pada versi aplikasi ini.

3. Berbagi data dengan pihak ketiga

Saat ini, Catet Dulu tidak membagikan data pribadi Anda kepada pihak ketiga. Jika di masa mendatang kami menambahkan layanan seperti iklan (misalnya Google AdMob), kebijakan ini akan diperbarui.

4. Pemeriksaan versi aplikasi

Aplikasi dapat memeriksa versi terbaru melalui file konfigurasi yang di-host secara publik untuk keperluan pembaruan wajib (force update). Pemeriksaan ini tidak mengirim data keuangan atau profil Anda.

5. Ekspor data

Anda dapat mengekspor laporan pengeluaran ke file Excel melalui fitur di aplikasi. File tersebut disimpan atau dibagikan sesuai tindakan Anda sendiri.

6. Hak pengguna

  • Anda dapat mengubah atau menghapus data dengan mengedit/menghapus transaksi di aplikasi.
  • Menghapus aplikasi dari perangkat akan menghapus data yang tersimpan secara lokal.

7. Keamanan

Kami berupaya menjaga data tetap aman di perangkat Anda. Namun, keamanan juga bergantung pada pengaturan perangkat Anda (kunci layar, backup, dan sebagainya).

8. Perubahan kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan material akan dicantumkan di halaman ini bersama tanggal pembaruan terbaru.

9. Kontak

Untuk pertanyaan terkait privasi, hubungi pengembang melalui halaman aplikasi di Google Play Store.
